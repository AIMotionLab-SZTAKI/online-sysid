# Simulation examples

To reproduce the results reported in the paper, run either the `batchwise_learning.py` file or `recursive_updates.py`.
The former implements the batch-wise model learning approach, while the latter can be used to run the recursive parameter update scheme.
Hyperparameters are set to match the ones used in the paper. Otherwise can be modified by editing the scripts.

All reported results were obtained using **Python 3.10** and **Ubuntu 24.04.3 LTS**.
